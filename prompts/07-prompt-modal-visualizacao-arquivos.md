# Prompt — Modal para Visualização dos Arquivos

Vamos resolver um último problema.

Quando clicarmos em "Visualizar arquivo", existe uma forma de mostrar o arquivo ou print que o cliente enviou?

Implemente um Modal de Pré-Visualização.

Comportamento:

- Imagens PNG/JPG: exibir o print em tamanho adequado.
- PDFs: exibir dentro de um iframe.
- Outros formatos: disponibilizar botão para abrir em nova aba.

O modal deve conter:

- Header com nome do arquivo.
- Categoria.
- Resumo gerado pela IA.
- Footer com data de envio.
- Botão "Abrir em nova aba".
- Botão "Fechar".

Mantenha o design integrado à interface existente.
