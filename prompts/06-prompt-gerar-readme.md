# Prompt — Gerar README do Projeto

Faça um resumo de tudo que foi feito no projeto e gere um README.md completo para o repositório.

O README deve conter:

- Missão do projeto.
- Requisitos atendidos com checkboxes.
- Tecnologias utilizadas.
- Justificativa do modelo Gemini escolhido.
- Estimativa de custo por documento, aproximadamente US$ 0,00015 por arquivo.
- Considerações sobre LGPD.
- Limitações conhecidas de escrita em disco em ambientes serverless.
- Instruções de instalação.
- Configuração das variáveis de ambiente.
- Como executar localmente.
- Como fazer o deploy.
